# Kaso Design

A social/banner card generator for Kaso brand assets.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Features

- Live preview for Twitter/X, LinkedIn, Facebook, Instagram, and OG sizes
- Theme, typography, layout, icon style, and detail controls
- PNG export at full target resolution
- Preset import/export as JSON
- Local autosave (restores last session)
